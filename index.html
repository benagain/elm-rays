A mobile raycasting hack in Elm

Kris Jenkins wrote a raycaster in Elm. I've just put it in an app.

You'll need Cordova and Elm.

npm install -g cordova

git clone https://github.com/benagain/elm-rays.git && cd elm-rays

npm install
cordova platform add android
cordova run android --
